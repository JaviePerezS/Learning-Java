# ⚙️ Fundamentos de Gestión de Datos

### 🏗️ ETL (Extract, Transform, Load)
Representa el proceso de integración donde la transformación de los datos se realiza antes de su carga en el sistema de destino para asegurar su calidad.

### 🚀 ELT (Extract, Load, Transform)
Representa el flujo de integración donde los datos se cargan en bruto y la transformación se ejecuta directamente en el destino aprovechando su potencia de cómputo.

### 📑 Metadatos
Representa la información descriptiva que define el contexto, las características técnicas y el origen de los activos de datos.

### 📖 Diccionario de Datos
Representa el catálogo formal que define la estructura técnica, los tipos de datos y las relaciones de los elementos dentro de una base de datos.

### 🛤️ Linaje de Datos
Representa la trazabilidad histórica que registra el origen, el recorrido y las transformaciones de un dato a través de toda la infraestructura.
