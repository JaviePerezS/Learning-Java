## Introducción
Este manual proporciona una guía paso a paso para instalar y configurar Elasticsearch y Kibana utilizando Podman, un motor de contenedores sin daemon que es compatible con Docker. Al final de este manual, tendrá un entorno funcional donde Elasticsearch y Kibana operan en contenedores gestionados por Podman.



### Requisitos Previos
    • Sistema Operativo: Linux (por ejemplo, CentOS, Fedora, RHEL)
    • Privilegios: Acceso como usuario root o con permisos sudo
    • Conexión a Internet: Para descargar las imágenes de los contenedores
    • JAVA 8 o Superior: Utilizar la JDK de Java 8 o superior.
