# ☕ Learning-Java

Bienvenido a **Learning-Java** 🚀  

Este repositorio tiene como propósito consolidar una **guía de aprendizaje de Java desde cero hasta un nivel avanzado (Zero to Hero)**, de manera clara, práctica e intuitiva.

---

## 🎯 Objetivo

Construir una ruta estructurada que permita:

- Aprender Java desde los fundamentos.
- Entender los conceptos clave de la Programación Orientada a Objetos.
- Aplicar buenas prácticas desde el inicio.
- Desarrollar proyectos prácticos.
- Evolucionar hacia conceptos avanzados y herramientas del ecosistema Java.

Este repositorio está pensado como una guía progresiva, donde cada tema construye sobre el anterior.

---

## 🛣️ Roadmap de Aprendizaje

### 🟢 Nivel 1 – Fundamentos

- ¿Qué es Java?
- Instalación del JDK
- Variables y tipos de datos
- Operadores
- Estructuras de control (if, switch)
- Ciclos (for, while, do-while)
- Métodos
- Arreglos

---

### 🟡 Nivel 2 – Programación Orientada a Objetos (POO)

- Clases y Objetos
- Encapsulamiento
- Herencia
- Polimorfismo
- Abstracción
- Constructores
- Modificadores de acceso
