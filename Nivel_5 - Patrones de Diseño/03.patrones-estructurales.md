# Patrones de Diseño Estructurales en Java

Los patrones estructurales explican cómo ensamblar objetos y clases en estructuras más grandes, a la vez que se mantiene la flexibilidad y eficiencia de estas estructuras.

---

## Patrón Adapter
**Propósito:** Permite que objetos con interfaces incompatibles colaboren.

```java
interface Target {
    void request();
}

class Adaptee {
    public void specificRequest() {
        System.out.println("Solicitud específica del Adaptee.");
    }
}

class Adapter implements Target {
    private Adaptee adaptee;

    public Adapter(Adaptee adaptee) {
        this.adaptee = adaptee;
    }

    @Override
    public void request() {
        adaptee.specificRequest();
    }
}
```

## Patrón Bridge
**Propósito:** Divide una clase grande en dos jerarquías separadas (abstracción e implementación) que pueden desarrollarse independientemente.

```java
interface Device {
    void setVolume(int percent);
}

class TV implements Device {
    public void setVolume(int percent) { System.out.println("TV volumen: " + percent); }
}

class RemoteControl {
    protected Device device;
    public RemoteControl(Device device) { this.device = device; }
    public void volumeUp() { device.setVolume(100); }
}
```

## Patrón Composite
**Propósito:** Permite componer objetos en estructuras de árbol y trabajar con esas estructuras como si fueran objetos individuales.

```java
import java.util.ArrayList;
import java.util.List;

interface Graphic {
    void draw();
}

class Leaf implements Graphic {
    public void draw() { System.out.println("Dibujando hoja."); }
}

class Composite implements Graphic {
    private List<Graphic> children = new ArrayList<>();
    public void add(Graphic g) { children.add(g); }
    public void draw() {
        for (Graphic g : children) g.draw();
    }
}
```

## Patrón Decorator
**Propósito:** Permite añadir funcionalidades a objetos colocando estos objetos dentro de objetos encapsuladores especiales.

```java
interface Coffee {
    String getDescription();
}

class SimpleCoffee implements Coffee {
    public String getDescription() { return "Café simple"; }
}

abstract class CoffeeDecorator implements Coffee {
    protected Coffee decoratedCoffee;
    public CoffeeDecorator(Coffee coffee) { this.decoratedCoffee = coffee; }
}

class MilkDecorator extends CoffeeDecorator {
    public MilkDecorator(Coffee coffee) { super(coffee); }
    public String getDescription() { return decoratedCoffee.getDescription() + " con leche"; }
}
```

## Patrón Facade
**Propósito:** Proporciona una interfaz simplificada a una biblioteca, un framework o cualquier otro grupo complejo de clases.

```java
class CPU { public void freeze() { System.out.println("CPU congelada."); } }
class Memory { public void load() { System.out.println("Memoria cargada."); } }

class ComputerFacade {
    private CPU cpu = new CPU();
    private Memory memory = new Memory();

    public void start() {
        cpu.freeze();
        memory.load();
    }
}
```

## Patrón Flyweight
**Propósito:** Permite mantener más objetos dentro de la cantidad disponible de memoria RAM compartiendo las partes comunes del estado.

```java
import java.util.HashMap;
import java.util.Map;

class TreeType {
    private String name;
    private String color;
    public TreeType(String name, String color) { this.name = name; this.color = color; }
}

class TreeFactory {
    private static Map<String, TreeType> treeTypes = new HashMap<>();
    public static TreeType getTreeType(String name, String color) {
        if (!treeTypes.containsKey(name)) treeTypes.put(name, new TreeType(name, color));
        return treeTypes.get(name);
    }
}
```

## Patrón Proxy
**Propósito:** Proporciona un sustituto o marcador de posición para otro objeto para controlar el acceso a él.

```java
interface Image { void display(); }

class RealImage implements Image {
    public void display() { System.out.println("Mostrando imagen real."); }
}

class ProxyImage implements Image {
    private RealImage realImage;
    public void display() {
        if (realImage == null) realImage = new RealImage();
        System.out.println("Log: Accediendo a la imagen.");
        realImage.display();
    }
}
```
